# 后台管理系统使用指南

## 🔒 安全访问

### 管理员密码
- 密码已移至环境变量保护
- 默认密码：`cyber_admin_2025_secure`（请立即修改）
- 在 `.env.local` 文件中修改 `VITE_ADMIN_PASSWORD`

### 安全提醒
- ⚠️ 请勿使用默认密码
- ⚠️ 定期更换管理员密码
- ⚠️ 保护 `.env.local` 文件安全

CyberBlog 现在包含了一个专业的后台管理系统，用于编辑和管理博客内容。

## 如何访问管理后台

1. 在主网站上，点击导航栏的 "设置" 选项卡
2. 在设置页面中，输入管理员密码：`chen1234`
3. 点击 "解锁控制台" 按钮
4. 系统将自动切换到专业的后台管理界面

## 管理后台功能

### 数据概览
- 查看总文章数、总项目数、总浏览量等统计信息
- 显示最近的活动记录

### 文章管理
- 查看所有文章的列表
- 支持新建文章
- 支持编辑现有文章
- 支持删除文章
- 显示文章标签、发布日期和浏览量

### 项目管理
- 类似文章管理，但针对项目内容
- 支持完整的CRUD操作

### 系统设置
- 网站标题设置
- 管理员邮箱设置
- 其他系统配置

## 技术特性

- **响应式设计**：支持桌面和移动设备
- **实时数据同步**：基于Firebase实时数据库
- **Markdown支持**：内容编辑支持Markdown格式
- **现代化UI**：采用赛博朋克风格设计
- **安全访问**：需要密码验证才能进入

## 注意事项

- 所有数据都会实时同步到Firebase云端
- 删除操作不可撤销，请谨慎操作
- 建议定期备份重要数据
- 密码仅用于演示，生产环境请使用更安全的认证方式
